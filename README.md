# FLETY 🚚

Plataforma de conexión entre clientes y Fletyers en Uruguay.

## Tecnología
- React 18 + Vite
- Sin dependencias externas adicionales

## Para desarrollar localmente
```bash
npm install
npm run dev
```

## Para construir para producción
```bash
npm run build
```
